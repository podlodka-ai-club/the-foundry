#  Team The Foundry. Architecture (Hacker Sprint, AI Club)

### Текущий прогресс проекта

- Александр создал рабочий pipeline для GitHub integration
  - Автоматически забирает issues с лейблом “agent task”
  - Создает pull requests и закрывает задачи
  - Интегрирован с Git, но агентская часть пока заглушка
- Майкл реализовал MVP агента с AIDR
  - Поддержка 3 моделей: DeepSeek, ChatGPT, Claude
  - Создает задачи в папке tasks, генерирует код
  - Работает через консольные команды и make-файл
  - Есть проблемы с контекстом файлов проекта

### Архитектура системы

- Pipeline состоит из 5 этапов:
  1. Fetch - забрать issue из трекера
  2. Context - собрать контекст для задачи
  3. Plan - создать план через модель
  4. Implement - реализация кода агентом
  5. Verify - проверка сборки и тестов
- Зеленые этапы уже работают (Fetch, PR creation)
- Нужно реализовать Context, Plan, Implement

### Технические детали

- Используется SQLite для статуса задач
- Тестовый репозиторий для обкатки
- Майкл работает через subprocess вызовы
- Александр предлагает SDK подход для удобства
- Возможна интеграция с Langfuse для логирования

### Планы интеграции

### Дополнительные возможности
