**Задача AI-1: Разработка скрипта aider/agent.py для автоматизации выполнения задач через AI-агента**

**Дата создания:** 23.04.2026

---

**Описание задачи**

Разработать Python-скрипт `aider/agent.py`, 
который будет оберткой над инструментом aider для автоматического выполнения задач по разработке с логированием результатов работы.

---

**Требования**

**1. Параметры запуска скрипта:**

- `--task` (обязательный) - номер задачи
- `--prompt` (необязательный) - текст задачи в виде промта

**2. Валидация номера задачи:**

Номер задачи может содержать:
- Цифры (например: `23453`)
- Латинские буквы, цифры и дефис (например: `GA-2`, `TF232`, `gq2342`)

Если номер задачи не указан или некорректный - выдать ошибку:
```
Ошибка: номер задачи не указан или некорректный. Проверьте значение параметра --task.
```

**3. Логика работы скрипта:**

**Случай 1:** Указан только `--task` (без `--prompt`)
```bash
python aider/agent.py --task="TF-1434"
```
- Искать файл задачи: `aider/tasks/TF-1434/TF-1434_task.md`
- Если найден → читать и выполнять задачу из файла
- Если не найден → выдать ошибку и завершить работу

**Случай 2:** Указаны `--task` и `--prompt`
```bash
python aider/agent.py --task="TF-2" --prompt="Напиши скрипт, выводящий в консоль 'Hello world'"
```
- Искать файл задачи: `aider/tasks/TF-2/TF-2_task.md`
- Если найден → читать содержимое и склеить с текстом из `--prompt`
- Если не найден → создать файл `aider/tasks/TF-2/TF-2_task.md` и записать туда текст из `--prompt`
- Передать итоговый текст задачи в aider

**4. Рабочая директория:**

Все изменения должны выполняться только в папке `code/` и её подпапках. Необходимо ограничить aider, чтобы он не мог изменять файлы вне этой директории.

**5. Настройки LLM:**

Скрипт должен читать настройки из `.env` файла:
```
# LLM settings
CODING_LLM=DEEPSEEK
DEEPSEEK_API_KEY=sk-22321bd****
ANTHROPIC_API_KEY=****
CHATGPT_API_KEY=****
```

Использовать LLM, указанную в переменной `CODING_LLM`, и соответствующий API ключ.

**6. Формат лога:**

Результат работы сохранять в файл: `aider/tasks/<task_id>/<task_id>_log.md`

Содержимое лога:
- Исходный промт (текст задачи)
- Полный вывод aider
- Комментарий для коммита

**7. Обработка ошибок:**

При ошибке выполнения задачи:
- Записать сообщение об ошибке в лог: `aider/tasks/<task_id>/<task_id>_log.md`
- Вывести в консоль:
```
Ошибка выполнения задачи. Лог работ сохранен в файл: aider/tasks/<task_id>/<task_id>_log.md
```

**8. Успешное выполнение:**

При успешном выполнении вывести в консоль:
```
Задача выполнена, лог работ сохранен в файл: aider/tasks/<task_id>/<task_id>_log.md
```

**9. Версия aider:**

Использовать последнюю стабильную версию `aider-chat`.

**Технические детали реализации**

**Валидация номера задачи:**
```python
import re

def validate_task_id(task_id: str) -> bool:
    pattern = r'^[A-Za-z0-9\-]+$'
    return bool(re.match(pattern, task_id)) and len(task_id) > 0
```

**Ограничение aider рабочей директорией:**
- Использовать параметр `--subtree-only` для ограничения контекста
- Запускать aider с `cwd=code/`
- Передавать в aider только файлы из `code/`
