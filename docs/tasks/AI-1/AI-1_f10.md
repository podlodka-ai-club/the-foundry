# AI-1_f10: Исправление видимости файлов в Aider

**Дата:** 2026-04-23 19:37

## Исходная задача

Изучить лог выполнения команды `wsl make runagent task=TF-2` и исправить проблему с видимостью файлов в aider.

## Проблема

При выполнении задачи TF-2 aider не видел существующий файл `code/hello.py`:

```
We need to see the current content of code/hello.py. 
Since the user hasn't added any files yet, we need to ask them to add the file to the chat.

Please add the file code/hello.py to the chat so I can see its current content 
and propose the necessary changes.
```

**Лог:** `aider/tasks/TF-2/TF-2_log.md`

**Задача TF-2:**
```
Доработай скрипт code/hello.py, 
чтобы при его запуске спрашивалось пользователя,
и выводилась фраза приветствия вместе с его именем.
```

**Результат:** Aider не смог выполнить задачу, так как не видел файл.

---

## Анализ проблемы

### Причина

В `AiderRunner.run()` файлы добавлялись в командную строку:

```python
existing_files = self._get_existing_files()
if existing_files:
    aider_cmd.extend(existing_files)
```

**Команда выглядела так:**
```bash
aider --yes-always --no-git hello.py factorial.py --message "доработай hello.py"
```

**Проблема:** Без флага `--read` aider **не читает** содержимое файлов в контекст чата.

### Дополнительная проблема

Метод `_get_existing_files()` возвращал только `.py` файлы:

```python
def _get_existing_files(self) -> List[str]:
    existing_files = list(self.code_dir.glob('*.py'))
    return [str(file.name) for file in existing_files]
```

**Проблема:** В папке `code/` может быть код на любом языке (JS, Go, Rust, etc.), не только Python.

---

## Решение

### 1. Добавлен флаг `--read`

**Файл:** `aider/core/aider_runner.py`, строка 54-66

**Было:**
```python
base_cmd = ['aider', '--yes-always', '--no-git']

aider_cmd, env = provider.configure_aider_command(base_cmd, env)

existing_files = self._get_existing_files()
if existing_files:
    aider_cmd.extend(existing_files)
```

**Стало:**
```python
# Базовая команда с флагами:
# --yes-always: автоматически подтверждать все действия
# --no-git: отключить git (мы управляем git вручную)
base_cmd = ['aider', '--yes-always', '--no-git']

# Провайдер добавляет свои параметры (--model, API ключи через env)
aider_cmd, env = provider.configure_aider_command(base_cmd, env)

# Добавляем --read перед списком файлов
# ВАЖНО: --read должен идти непосредственно перед файлами
existing_files = self._get_existing_files()
if existing_files:
    aider_cmd.append('--read')
    aider_cmd.extend(existing_files)
```

**ВАЖНО:** Порядок аргументов критичен! Флаг `--read` должен идти **непосредственно перед** списком файлов, после всех остальных опций (`--model`, etc.).

**Что делает `--read`:**
- Добавляет указанные файлы в контекст aider **для чтения**
- Aider видит содержимое файлов
- Может предлагать изменения в этих файлах
- Файлы доступны только для чтения (не для прямого редактирования)

**Пример команды:**
```bash
aider --yes-always --no-git --model deepseek/deepseek-chat --read hello.py factorial.py --message "доработай hello.py"
```

### 2. Изменена логика получения файлов

**Файл:** `aider/core/aider_runner.py`, строка 94-107

**Было:**
```python
def _get_existing_files(self) -> List[str]:
    """
    Получение списка существующих файлов для контекста aider.
    
    Returns:
        Список имен файлов .py в директории code
    """
    existing_files = list(self.code_dir.glob('*.py'))
    return [str(file.name) for file in existing_files]
```

**Стало:**
```python
def _get_existing_files(self) -> List[str]:
    """
    Получение списка существующих файлов для контекста aider.
    
    Возвращает все файлы в директории code (любые расширения),
    исключая служебные файлы aider (.aider.*).
    
    Returns:
        Список имен файлов в директории code
    """
    all_files = [f for f in self.code_dir.iterdir() if f.is_file()]
    # Исключаем служебные файлы aider
    existing_files = [f for f in all_files if not f.name.startswith('.aider')]
    return [str(file.name) for file in existing_files]
```

**Изменения:**
- ✅ Возвращаются **все файлы** (не только `.py`)
- ✅ Исключаются служебные файлы aider (`.aider.*`)
- ✅ Поддержка любых языков программирования

**Исключаемые файлы:**
- `.aider.chat.history.md` - история чата aider
- `.aider.input.history` - история ввода
- `.aider.tags.cache.v4` - кэш тегов
- `.aider.model.metadata.json` - метаданные модели (НЕ исключается, так как не начинается с `.aider`)

**Включаемые файлы:**
- `*.py` - Python
- `*.js` - JavaScript
- `*.go` - Go
- `*.rs` - Rust
- `*.java` - Java
- `*.cpp`, `*.c`, `*.h` - C/C++
- `*.md` - Markdown
- И любые другие файлы

## Дополнительные улучшения (будущее)

### 1. Фильтрация по расширениям

Добавить в `.env` настройку для фильтрации файлов:

```env
# Расширения файлов для добавления в контекст aider (через запятую)
# Пустое значение = все файлы
AIDER_FILE_EXTENSIONS=py,js,go,rs,java
```

**Код:**
```python
def _get_existing_files(self) -> List[str]:
    extensions = os.getenv('AIDER_FILE_EXTENSIONS', '')
    if extensions:
        allowed_exts = set(ext.strip() for ext in extensions.split(','))
        all_files = [f for f in self.code_dir.iterdir() 
                     if f.is_file() and f.suffix[1:] in allowed_exts]
    else:
        all_files = [f for f in self.code_dir.iterdir() if f.is_file()]
    
    existing_files = [f for f in all_files if not f.name.startswith('.aider')]
    return [str(file.name) for file in existing_files]
```

### 2. Ограничение размера файлов

Исключать слишком большие файлы из контекста:

```python
MAX_FILE_SIZE = 1024 * 1024  # 1 MB

def _get_existing_files(self) -> List[str]:
    all_files = [f for f in self.code_dir.iterdir() 
                 if f.is_file() and f.stat().st_size < MAX_FILE_SIZE]
    existing_files = [f for f in all_files if not f.name.startswith('.aider')]
    return [str(file.name) for file in existing_files]
```

### 3. Приоритизация файлов

Добавлять сначала файлы, упомянутые в задаче:

```python
def _get_existing_files(self, task_text: str = '') -> List[str]:
    all_files = [f for f in self.code_dir.iterdir() if f.is_file()]
    existing_files = [f for f in all_files if not f.name.startswith('.aider')]
    
    # Приоритизация файлов, упомянутых в задаче
    mentioned = [f for f in existing_files if f.name in task_text]
    others = [f for f in existing_files if f.name not in task_text]
    
    return [str(f.name) for f in mentioned + others]
```
